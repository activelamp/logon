Having trouble signing in?

You can reset your password online if you have a security question set, or lookup your UCLA Logon ID if you don't remember it.

If you're sure you don't have one, go ahead and create a UCLA Logon ID.

The UCLA Logon ID is your campus online identifier. It is used for authentication to most online services on campus. It also provides eligible users with access to Bruin OnLine services such as an e-mail account, network access, web hosting service, etc. Previously, the UCLA Logon ID was called a Bruin OnLine ID or BOLid.
