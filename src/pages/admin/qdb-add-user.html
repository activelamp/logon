Create a QDB User Account

To add an existing UCLA employee to the QDB, perform the following:

  1. Enter the 9-digit UCLA employee ID into the "Add Employee ID" form field. Click the Submit button.
  2. If the UCLA employee ID is valid, then the employee's loginID (and other details) will be displayed.
    • If the employee ISN'T already a QDB user , then click the loginID link to add him/her to theQDB.
    • Otherwise if the employee IS already a QDB user a message indicating QDB membership will be displayed instead.
